## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript sample.

* Nest.js + [Cloud Spanner](https://cloud.google.com/spanner)
* with Cloud Spanner thin orm
* with Docker
* with Cloud Spanner emulator

## Installation

```bash
$ yarn 
```

## Running the app

```bash
# development
$ docker-compose down
$ docker-compose up

# watch mode(local)
$ yarn start:dev
```
